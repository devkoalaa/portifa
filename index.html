Sobre Mim

Uma breve introdução

Olá! Sou Herbert, um desenvolvedor apaixonado por criar aplicações que resolvem problemas reais e proporcionam ótimas experiências para o usuário. Com um sólido background em desenvolvimento full-stack, meu foco é na construção de sistemas eficientes e de alta qualidade.

Minha jornada na programação começou com o desejo de transformar ideias em realidade digital. Hoje, trabalho com um stack moderno centrado em TypeScript, utilizando React e React Native no front-end e Node.js com NestJS no back-end.

Estou sempre em busca de novos desafios e aprendizados, explorando tecnologias como Docker para otimizar ambientes de desenvolvimento e produção.
